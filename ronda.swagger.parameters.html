ronda.swagger.parameters

collect

(collect schema request-method route-params)

Inputs: [schema :- rq/RawRequestSchema request-method :- s/Keyword route-params :- [s/Any]] Returns: sw/Parameters

Collect all parameters from the given schema.

  • if the method is GET/HEAD/OPTIONS, :params will be interpreted as query parameters,
  • if a body is given, :params will be interpreted as query parameters,
  • otherwise, :params will be interpreted as form parameter.

:headers and :body will be used directly.
